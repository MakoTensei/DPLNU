 The process begins with the 'Regression report' start event, which triggers the 'Send payment request to VN' task. This task is connected to 'ExclusiveGateway0', which then leads to 'Set process for resubmission'. The 'Set process for resubmission' task is linked to 'IntermediateCatchEvent4', which in turn leads to 'EndEvent2'.

Meanwhile, 'ExclusiveGateway0' also connects to 'ExclusiveGateway2', which receives the 'Examination' task. This task is linked to 'EndEvent2'. Additionally, 'ExclusiveGateway0' is also connected to 'EventBasedGateway3', which receives the 'IntermediateCatchEvent5' task. This task leads to 'EndEvent0'.

The 'Send payment request to VN' task is also connected to 'IntermediateCatchEvent6', which leads to 'EndEvent1'. Lastly, 'EndEvent0' and 'EndEvent1' are both connected to 'EndEvent2', which is the final event in the process.
