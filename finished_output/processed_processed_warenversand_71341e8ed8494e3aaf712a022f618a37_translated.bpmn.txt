 The Secretariat clears shipping by receiving incoming data from ParallelGateway4 and sending outgoing data to ExclusiveGateway0. This process is followed by the Filling the package note task, which also receives data from ParallelGateway4 and sends outgoing data to ExclusiveGateway2. The Pick up offers task then receives data from ExclusiveGateway0 and sends outgoing data to Select your offer. The Logistics manager completes insurance, which also receives data from Insurance necessary and sends outgoing data to ExclusiveGateway2. Warehouse workers and Packaging goods tasks receive data from ParallelGateway4 and send outgoing data to EndEvent0. The Select your offer task receives data from ExclusiveGateway0 and sends outgoing data to Specialist. The Specialist task receives data from ExclusiveGateway0 and sends outgoing data to ExclusiveGateway2. The ExclusiveGateway0 receives incoming data from ParallelGateway4 and sends outgoing data to Filling the package note, Specialist, and EndEvent0. Insurance necessary receives incoming data from ParallelGateway4 and sends outgoing data to ExclusiveGateway2 and ParallelGateway3. ExclusiveGateway2 receives incoming data from Insurance necessary, StartEvent0, and Specialist, and sends outgoing data to ParallelGateway3. ParallelGateway3 receives incoming data from ParallelGateway4 and ParallelGateway4, and sends outgoing data to EndEvent0. StartEvent0 receives no incoming data and sends outgoing data to ExclusiveGateway2. EndEvent0 receives incoming data from Packaging goods and sends no outgoing data.
