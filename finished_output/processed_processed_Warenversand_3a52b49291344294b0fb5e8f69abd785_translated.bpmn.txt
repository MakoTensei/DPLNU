 In the bustling world of logistics, a series of tasks unfolds to ensure the seamless delivery of goods. The journey begins with the initiation event, 'Goods to be sent', which triggers a cascade of operations.

Firstly, the 'Create a quote' task is activated, receiving three offers to select the most suitable one. This task then connects to 'Select a quote & order Spediteur', which is responsible for choosing the best quote and coordinating with the shipping service, Spediteur.

Simultaneously, the 'Pick up and ship goods' task is triggered, receiving forward goods to be picked up. It then proceeds to 'Goods shipped', marking the successful dispatch of the goods.

The 'Filling the package note' task is also initiated, receiving package note details to be filled. This task links to 'ParallelGateway6', which further connects to 'Providing forward goods', ensuring the package is ready for shipment.

Meanwhile, the 'Pick up and post goods' task is activated, receiving packageware to be picked up and posted. It then connects to 'Goods shipped', indicating the successful delivery of the packageware.

The 'Complete insurance' task is also triggered, receiving package details to complete insurance coverage. It connects to 'InclusiveGateway4', which further connects to 'ParallelGateway6', ensuring the package is insured and ready for shipment.

The 'Packaging goods' task is initiated, receiving goods to be packaged. It connects to 'ParallelGateway6', which further connects to 'Goods shipped', indicating the successful packaging and delivery of the goods.

The 'Providing forward goods' task is also activated, receiving package details to be forwarded. It connects to 'ExclusiveGateway1', which further connects to 'Spediteur?', ensuring the package is forwarded to the shipping service.
